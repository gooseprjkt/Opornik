# Опорник

Опорник - это библиотека правил русского языка, разработанная на Flutter с использованием Material Design 3 и динамических цветов. Приложение предоставляет доступ к базе данных правил русской орфографии и пунктуации.

## Особенности

- Material Design 3 с динамическими цветами
- Кроссплатформенная поддержка (Android, iOS, Web)
- Полнотекстовый поиск по правилам
- Иерархическая навигация: Разделы → Главы → Параграфы → Правила
- Поддержка форматированного отображения текста с выделениями
- Локализация на русском языке

## Архитектура

Проект использует следующую структуру:

- `lib/models/` - Модели данных, соответствующие структуре JSON файлов
- `lib/database/` - Слой доступа к данным из JSON файлов
- `lib/screens/` - Экраны приложения с навигацией
- `lib/utils/` - Вспомогательные утилиты, включая форматирование разметки

## Запуск приложения

1. Убедитесь, что у вас установлен Flutter
2. Клонируйте репозиторий
3. Выполните `flutter pub get`
4. Запустите приложение с помощью `flutter run`

## Поддержка веб-версии

Приложение полностью поддерживает веб-платформу благодаря использованию sqflite, который автоматически включает поддержку web через sqflite_web при сборке для браузера.

## Данные

Приложение использует JSON файлы для хранения данных правил русского языка. Данные содержат:
- Разделы (Орфография, Пунктуация)
- Главы (например, "Правописание гласных", "Правописание согласных")
- Параграфы с описаниями
- Правила с аннотациями и содержанием

## Технологии

- Material Design 3
